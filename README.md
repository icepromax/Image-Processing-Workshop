# Image-Processing-Workshop
本项目利用基础框架vue+fastapi实现，基于gan的运用，主要讲述图像处理方面
集成了三种模型，实现图像风格迁移，图像高清化和图像修复。
并且附有gan的学习资料。

对于后端所用模型，可在网站查看模型介绍进行下载或者训练，训练代码上传。
