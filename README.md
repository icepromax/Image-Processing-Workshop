# Image-Processing-Workshop
本项目基于vue3+fastapi实现，基于gan的运用，主要讲述图像处理方面
集成了三种模型，可体验实现图像风格迁移，图像高清化和图像修复。并且给出了gan的学习资源和大量相关论文链接，可供学习和参考。

对于后端所用模型，可在网站查看模型介绍进行下载或者训练，训练代码上传。

<h3>🌟 项目亮点</h3>

🖼️ 集成三大图像处理模型
🚀 提供完整的训练代码
📚 精选GAN学习资源库
🎯 开箱即用的在线体验

<h3>🛠️ 快速开始</h3>

前端

cd frontend<br/>
npm install

后端

cd backend<br/>
pip install -r requirements.txt
